# Logos e Imagens de Produtos

Logos oficiais das ferramentas e softwares avaliados no site.

- Formato: **PNG com fundo transparente** ou SVG
- Tamanho sugerido: **200×200px** (quadrado)
- Nomenclatura: `[produto]-logo.png`

## Exemplos
```
notion-logo.png
hubspot-logo.png
clickup-logo.png
canva-logo.png
pipedrive-logo.png
rdstation-logo.png
```

> Use logos oficiais obtidos dos kits de imprensa de cada empresa.
