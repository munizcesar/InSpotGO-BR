# Banners do Site

## hero/
Banners para as hero sections das páginas principais.
- Formato: **PNG ou WebP**, proporção **16:5** (~1400×440px)
- Exemplos: `home-hero.png`, `saas-hero.png`, `software-hero.png`

## categorias/
Banners de destaque por categoria, usados em topo de páginas de listagem.
- Formato: **PNG ou WebP**, proporção **3:1** (~900×300px)
- Nomenclatura: `banner-saas.png`, `banner-software.png`, `banner-guias.png`, `banner-comparativos.png`

## afiliados/
Banners promocionais de programas de afiliados.
- Formato: **PNG ou WebP**
- Tamanhos comuns: 728×90, 300×250, 320×100 (mobile)
- Nomenclatura: `banner-hubspot-728x90.png`, `banner-canva-300x250.png`
